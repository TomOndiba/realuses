Web File Browsing

When it comes to the overall style and functionality of the Web Access file viewing web app, I was really focusing on trying to create an experience that was consistent across platforms and devices. To me, that meant looking at existing structure and style frameworks that my developers were already using and familiar with. In our case, Twitter’s Bootstrap framework for the web, and creating similar styles as I had used in designing the mobile application’s for iOS and Android devices.

Mobile Apps

If the screenshots remind you of an application you’ve used, I would say you have a good eye. Pat yourself on the back. The Ark site was originally designed to create a familiar interface for new Ark users using a Facebook look. My goal in creating the mobile application for Ark was to replicate as much functionality and look of the site as I could. I looked to the robust Facebook iPhone app for inspiration.
Other design elements of the app were inspired by some great designers including Mark Jardine (@markjardine) of Tapbots, Sean Kovacs’s (@seankovacs) GV Mobile, Evernote and several others.

Wikipedia Search Redesign

  • Homepage Redesign

  • Main Page Redesign

  • Article Page Redesign

  • Search Results Redesign

This redesign was inspired by the Wikipedia Search Design Contest.

This contest is about redesigning Wikipedia's search experience from the ground up: everything from the behaviors on the search page, to the layout of content, to the details of typography, color, animations. Today, many people use Google to search Wikipedia, but imagine a whole new experience that makes using Google to search Wikipedia feel outdated.

Although, I didn't enter in to the contest, I decided to give it a try anyway.
