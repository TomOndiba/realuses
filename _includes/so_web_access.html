Web File Browsing

When it comes to the overall style and functionality of the Web Access file viewing web app, I was really focusing on trying to create an experience that was consistent across platforms and devices. To me, that meant looking at existing structure and style frameworks that my developers were already using and familiar with. In our case, Twitter’s Bootstrap framework for the web, and creating similar styles as I had used in designing the mobile application’s for iOS and Android devices.
